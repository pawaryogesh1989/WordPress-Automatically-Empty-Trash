Use this plugin to schedule Automatically Empty the Trash after set number of days.

== Description ==
 
This plugin can be used Automatically Empty Trash after the number of days the user requires. User can set the number of days from which the trash should be automatically deleted in the plugin option in the Admin section.

= Details =

* Option to set number of days.
* Automatically Empty Trash after number of days.

== Installation ==

The Plugin can be installed in two ways.

= Download and Install =

1. Download and upload the plugin files to the `/wp-content/plugins/automatically-empty-trash` directory from the Wordpress plugin repository.
2. Activate the plugin through the 'Plugins' screen in WordPress admin.

= Using Plugin Interface =

1. Open the plugin Interface in the wordpress admin and click on "Add new".
2. In the Search box enter "Automatically Empty Trash" and hit Enter.
3. Click on "Install" to install the plugin.
